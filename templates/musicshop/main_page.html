Инструкция по работе с системой

"Strings of the soul" - cистема управления заказами магазина музыкальных инструментов. Система позволяет добавлять, отслеживать, управлять заказами покупателей, вести каталогизацию товаров, отслеживать их наличие и расположение. Также система позволяет вести профили покупателей, отслеживать их заказы. Право создания заказов имеет сотрудник с уровнем доступа "Консультант" и выше. Доступ на сайт контролируется системой авторизации, каждый сотрудник имеет свой персональный профиль. Для создания заказа сотруднику следует переключиться на пункт меню "Каталог", где выбрать необходимую категорию. В открывшемся окне каталога будет таблица с доступными к продаже товарами и кнопками управления для добавления/удаления товара из корзины. Также в заголовке таблицы присутствуют элементы управления для сортировки, поиска по ключевым словами и группового добавления/удаления товаров. Стоит обратить внимание, что кнопка "Удалить отмеченные" удалит все отмеченные "галочкой" товары из корзины, вне зависимости от их количества в ней. Для более подробного описания каждого товара сотрудник может нажать кнопку "Подробнее" и откроется окно с данными о товаре, включая его изображние и распределение по складам. После добавления всех необходимых товаров, сотрудник может оформить заказ, кликнув по кнопке в правом верхнем углу с изображением торговой тележки и краткой информацией о товарах в корзине. Откроется страница оформления заказа, страница реализована на двух выпадающих элементах "Корзина" и "Оформление заказа, для удобства при оформлении больших заказов сотрудник может свернуть каждый из элементов. В элементе "Корзина" представляется таблица с товарами в корзине с теми же элементами управления, что и на странице каталога. В элементе "Оформление заказа представлена форма оформления заказа. Сотрудник должен выбрать покупателя из списка доступных профилей (при начале набора имени система произведет поиск по списку). Если покупатель не зарегистрирован в системе, то под полем выбора клиента есть ссылка на регистрацию нового покупателя. При нажатии на ссылку, слева появится еще одно окно формы для регистрации покупателя, по завершении заполнения, обновится страница корзины, и новый покупатель появится в списке. Поле "Сотрудник" является нередактируемым, имя сотрудника берется из текущей авторизованной сессии. После заполнения полей и подтверждения создания заказа, при успешном выполнении сотрудник окажется на странице, где будет сообщено о выполнении заказа и предложено скачать созданный документ (в формате pdf) по заказу.
